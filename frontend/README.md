# Frontend

## Stack

- Vue 3
- Vite
- TypeScript
- ESLint + Prettier

## Setup

```powershell
copy .env.example .env
pnpm install
pnpm dev
```

## Commands

- `pnpm dev`
- `pnpm build`
- `pnpm lint`
- `pnpm format`
- `pnpm format:check`

## Environment

- `VITE_API_BASE_URL`: backend API base URL, default `http://localhost:8000/api/v1`
